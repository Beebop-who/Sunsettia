 in settings (Edit > Project settings > Player > Other Settings) 
   - Color Space 'Gamma' change to 'Linear';

 You need download and install in you project asset "Wispy Skybox"
 (Url: https://assetstore.unity.com/packages/2d/textures-materials/sky/wispy-skybox-21737)
next in the settings you need to put
 (Window > Lighting > Environment Lighting)
   - Skybox = 'WispySkyboxMat';
   - Ambient Sourse = Skybox;
   - Ambient Intesity = 2;
    
 (Edit > Project settings > Quality > Rendering)
   - Pixel Light Count = 15;
   - Quality = Fantastic;
